These are kinds of attributes in html

  • lang attribute.

    The word lang is stands for language, it being used to define a website's language
    commonly lang attributes placed in <html>.
    for example :
    <html lang:"en-Us">

  • href attribute.

    The href attribute is used for defining a link, to tell where the website must to go if it's clicked. href is defined in anchor <a> element. for example :
    <a href="https://www.google.com">Click me to Google</a>
    will resulting this : Click me to Google

  • title attribute.
